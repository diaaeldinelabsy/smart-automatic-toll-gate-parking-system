# Smart Automatic Toll Gate & Car Parking System Using Arduino & RFID

This project is an automated toll gate and parking management system using Arduino, RFID authentication, IR sensors, a servo motor, an OLED display, LEDs, and a buzzer.

## Features

* RFID-based vehicle authentication
* Automatic gate opening and closing
* Vehicle detection using IR sensors
* Real-time parking occupancy monitoring
* OLED display for parking status and system messages
* Traffic light indicators (Red, Yellow, Green)
* Audio feedback using a buzzer

## Hardware Used

* Arduino Uno
* RFID RC522 Reader
* RFID Tags/Cards
* IR Sensors
* SG90 Servo Motor
* SSD1306 OLED Display
* LEDs
* Buzzer
* Breadboard and Jumper Wires
* 5V Power Supply

## Software

* Arduino IDE
* C/C++
* Libraries:

  * SPI
  * MFRC522
  * Servo
  * Wire
  * Adafruit_GFX
  * Adafruit_SSD1306

## System Operation

* Detects vehicles at the entry gate using an IR sensor
* Verifies RFID cards before granting access
* Opens the gate automatically for authorized vehicles
* Updates parking occupancy in real time
* Displays parking information on the OLED screen
* Detects exiting vehicles and updates available spaces
